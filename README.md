# 🔗 URL Shortener Backend API

A **RESTful Backend API** for a URL Shortener system built using **Node.js, Express.js, and MongoDB Atlas**.

This project was developed as part of an internship submission to demonstrate backend development skills including API design, database integration, URL validation, and analytics tracking.

---

# 📌 Project Objective

The main goal of this project is to convert long URLs into short, unique URLs and redirect users to the original link when accessed.

It also tracks usage analytics such as click counts for each shortened URL.

---

# ⚙️ Tech Stack

- **Runtime:** Node.js  
- **Framework:** Express.js  
- **Database:** MongoDB Atlas  
- **ODM:** Mongoose  
- **Tools:** Postman (for API testing)

---

# 🚀 Features

## 🔗 URL Shortening
- Converts long URLs into short, unique codes
- Generates easy-to-share links

## 🔁 Redirection System
- Redirects short URL to original long URL
- Handles invalid or non-existing URLs safely

## 📊 Click Analytics
- Tracks number of times each short URL is accessed
- Provides statistics for every shortened link

## 🚫 Duplicate Prevention
- Prevents storing duplicate long URLs
- Improves database efficiency

## 🔐 Data Validation
- Validates URL format before processing
- Ensures only valid URLs are stored
